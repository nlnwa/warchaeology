# warchaeology

Warchaeology is a collection of tools for inspecting, manipulating and validating WARC-files.

## Installation

[Installation instructions](https://nationallibraryofnorway.github.io/warchaeology/installation).

## Documentation

Usage information is available with the help command:

    warc help

Complete documentation can be found [here](https://nationallibraryofnorway.github.io/warchaeology/)
