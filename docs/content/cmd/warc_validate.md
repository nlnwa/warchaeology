## warc validate

Validate warc files

```
warc validate <files/dirs> [flags]
```

### Options

```

  -h, --help               help for validate
  -r, --recursive          walk directories recursively
      --suffixes strings   filter files by suffixes (default [.warc,.warc.gz])
  -s, --symlinks           follow symlinks
```
